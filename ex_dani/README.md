# Exercícios de Java - POO 🚀

Este repositório contém as atividades práticas desenvolvidas para a disciplina de Programação Orientada a Objetos na **Uni-FACEF**.

## 📂 Conteúdo do Projeto

1. **Sistema de Conta Bancária**: 
   - Modelagem de conta com operações de depósito, saque e encerramento.
   - Validação de saldo e status da conta.
2. **Sistema de Streaming**:
   - Gerenciamento de perfis (Básico, Premium e Família).
   - Cálculo automático de mensalidade e controle de status de assinatura.

## 🛠️ Tecnologias Utilizadas
* Java JDK
* IntelliJ IDEA
* Git/GitHub
